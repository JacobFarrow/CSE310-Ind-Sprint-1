C++ Personal Budget Tracker

This is a console-based application designed to help users track their spending by adding transactions, categorizing expenses, and monitoring their progress against a set budget limit. It features data persistence, saving all information to a CSV file so your records aren't lost when you exit.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open your terminal or command prompt.
2. Compile the program using a C++ compiler.
3. Run the executable by typing budget_tracker.exe on Windows.

Instructions for using the software:

1. Use the menu options (1-5) to add transactions or view your current spending.
2. Set a budget limit to see a live comparison of your total expenses versus your goal.
3. Select "Save and Exit" to ensure your data is written to budget_data.csv for next time.
